
java SystemLogin


Features
--------
1. Login Page:
- Input your name, user ID, email, and password.
- Validation ensures all fields are properly filled before proceeding.
- Displays a welcome message upon successful login.

2. Booking Selection Page:
- Choose between "Human Reservations" and "Remote Reservations."

3. Reservation Pages:
- Human reservations include options like Career Services and Peer Tutoring.
- Remote reservations include Housing Selection and Classroom Booking.
- Each option redirects to the appropriate URL in your default browser.


Example Commands
----------------
Compile: javac SystemLogin.java
Run: java SystemLogin
